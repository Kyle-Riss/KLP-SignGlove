# 훈련 (Training)

이 폴더는 SignGlove 모델의 훈련 관련 파일들을 포함합니다.

## 파일 목록

### `improved_training.py`
- **기능**: 개선된 모델 훈련 시스템
- **사용법**: `python3 improved_training.py`
- **입력**: `real_data_filtered/` 폴더의 정규화된 데이터
- **출력**: `models/improved_regularized_model.pth`

## 훈련 설정

### 모델 설정
- **모델 타입**: RGRU
- **입력 크기**: 8 (5개 Flex + 3개 IMU)
- **은닉 크기**: 96
- **클래스 수**: 24
- **드롭아웃**: 0.5

### 최적화 설정
- **옵티마이저**: AdamW
- **학습률**: 0.001
- **가중치 감쇠**: 0.01
- **배치 크기**: 32
- **에포크**: 100

### 정규화 설정
- **LayerNorm**: 입력 및 중간 레이어
- **Dropout**: 0.5
- **Gradient Clipping**: 1.0
- **Early Stopping**: 검증 손실 기준

## 사용 방법

```bash
# 훈련 실행
python3 training/improved_training.py
```

## 훈련 과정

1. **데이터 로드**: `real_data_filtered/`에서 정규화된 데이터 로드
2. **데이터 분할**: 훈련 60%, 검증 20%, 테스트 20%
3. **모델 초기화**: RGRU 생성
4. **훈련 루프**: 에포크별 훈련 및 검증
5. **모델 저장**: 최고 성능 모델 저장

## 훈련 결과

- **최종 검증 정확도**: 94.78%
- **최종 검증 손실**: 0.1852
- **과적합 지수**: 0.1766 (매우 낮음)
- **수렴 에포크**: 54

## 주요 특징

- **과적합 방지**: 다양한 정규화 기법 적용
- **학습률 스케줄링**: ReduceLROnPlateau 사용
- **조기 종료**: 검증 손실 기준
- **모델 체크포인트**: 최고 성능 모델 저장
- **훈련 곡선**: 실시간 성능 모니터링
