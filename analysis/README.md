# 분석 (Analysis)

이 폴더는 SignGlove 모델의 분석 관련 파일들과 결과를 포함합니다.

## 파일 목록

### `analyze_inference_accuracy.py`
- **기능**: 추론 정확도 분석
- **사용법**: `python3 analyze_inference_accuracy.py`
- **입력**: `corrected_filtered_results.json`
- **출력**: 정확도 분석 결과

### `real_training_curves.py`
- **기능**: 학습 곡선 생성
- **사용법**: `python3 real_training_curves.py`
- **입력**: 훈련된 모델 체크포인트
- **출력**: 학습 곡선 이미지

### `corrected_filtered_results.json`
- **기능**: 최종 추론 결과
- **내용**: 575개 파일의 추론 결과
- **평균 신뢰도**: 86.3%

### `real_training_curves.png`
- **기능**: 학습 곡선 시각화
- **내용**: 손실, 정확도, 과적합 분석
- **해상도**: 고해상도 (300 DPI)

### `real_training_report.txt`
- **기능**: 훈련 보고서
- **내용**: 훈련 과정 요약 및 성능 지표

## 사용 방법

### 추론 정확도 분석
```bash
# 정확도 분석 실행
python3 analysis/analyze_inference_accuracy.py
```

### 학습 곡선 생성
```bash
# 학습 곡선 생성
python3 analysis/real_training_curves.py
```

## 분석 결과

### 전체 성능
- **평균 신뢰도**: 86.3%
- **처리 속도**: 0.7ms
- **총 파일 수**: 575개

### 신뢰도 분포
- **높은 신뢰도 (>80%)**: 424개 (73.7%)
- **중간 신뢰도 (60-80%)**: 92개 (16.0%)
- **낮은 신뢰도 (<60%)**: 59개 (10.3%)

### 클래스별 성능
- **높은 정확도 (>90%)**: 11개 클래스
- **중간 정확도 (80-90%)**: 6개 클래스
- **낮은 정확도 (<80%)**: 7개 클래스

## 주요 특징

### 추론 정확도 분석
- **파일명 기반 정확도**: 실제 클래스와 예측 비교
- **클래스별 성능**: 각 클래스의 정확도 분석
- **혼동 행렬**: 클래스 간 혼동 패턴 분석
- **신뢰도 분석**: 정확/오답별 신뢰도 분포

### 학습 곡선 분석
- **손실 곡선**: 훈련/검증 손실 변화
- **정확도 곡선**: 훈련/검증 정확도 변화
- **과적합 분석**: 과적합 지수 계산
- **수렴 분석**: 모델 수렴 시점 확인

## 결과 파일 형식

### JSON 결과 파일
```json
{
  "file": "파일경로",
  "predicted_class": "예측클래스",
  "confidence": 0.863,
  "inference_time": 0.0007,
  "enhanced": true,
  "improved_model": true
}
```

### 학습 곡선 이미지
- **2x2 서브플롯**: 손실, 정확도, 로그 스케일, 과적합
- **색상 구분**: 훈련(파랑), 검증(빨강)
- **최고 성능 지점**: 수직선으로 표시
